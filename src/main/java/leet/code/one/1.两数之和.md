

两数之和
====


## 题目描述
给定一个*整数数组* `nums` 和一个*目标值* `target`，
请你在该数组中找出`和为目标值`的那 **两个** *整数*，并返回他们的`数组下标`。

你可以假设`每种输入只会对应一个答案`。但是，你`不能重复利用这个数组中同样的元素(元素值与下标都相同)`。

示例：
```
给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]
```


## 题解
> 官方题解，哈希表/散列表/注册表

### 方法一：暴力法
复杂度分析：
* 时间复杂度：O(n^2)
* 空间复杂度：O(1)

### 方法二：两遍哈希表
复杂度分析：
* 时间复杂度：O(n)
* 空间复杂度：O(n)
